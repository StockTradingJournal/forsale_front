# ForSale 게임 프론트엔드

ForSale 보드게임의 프론트엔드 클라이언트입니다.

## 🚀 빠른 시작

### macOS/Linux 사용자

1. **환경 설정 (최초 1회만)**
   ```bash
   ./setup.sh
   ```

2. **프론트엔드 실행**
   ```bash
   ./start.sh
   ```

## 📋 수동 실행 방법

환경 설정:
```bash
npm install
```

프론트엔드 실행:
```bash
npm run dev
```

## 🌐 접속 정보

- **프론트엔드 주소**: http://localhost:5173
- **백엔드 연결**: http://localhost:8000

## 🛠 개발 환경

- Node.js 18+
- React 18
- Vite
- TypeScript
- Socket.IO Client
